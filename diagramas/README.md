# Diagramas de flujo

En esta carpeta se almacenan los diagramas de flujo elaborados para representar de forma visual la lógica de funcionamiento del programa desarrollado en el proyecto. Estos diagramas muestran la secuencia de procesos, validaciones, toma de decisiones y resultados de las principales funcionalidades del sistema.

## Diagramas incluidos

- **Diagrama de Flujo del Menú**  
  Representa el flujo general del menú del programa, desde el inicio, la visualización de opciones, la lectura de la opción ingresada y la validación correspondiente. También muestra el tratamiento de errores y el retorno al menú o finalización del proceso.

- **Registro o Ingreso de Datos**  
  Muestra el proceso de solicitud, lectura y validación de datos ingresados por el usuario. En caso de error, el sistema vuelve a solicitar la información hasta que los datos sean correctos, para luego procesarlos y mostrar el resultado.

- **Proceso Principal**  
  Describe el flujo de ejecución principal del sistema, incluyendo el ingreso de valores, su validación, la realización del cálculo correspondiente y la presentación del resultado final o del mensaje de error.

## Finalidad

Estos diagramas fueron desarrollados con el objetivo de facilitar la comprensión de la lógica del programa antes y durante la etapa de codificación, permitiendo una mejor organización del desarrollo del software.
